# AI Engineering — Learning & Interview Prep

A structured curriculum to become an expert AI Engineer and prepare for interviews. Each folder is a self-contained study guide with concepts, interview questions, hands-on tasks, and resources.

> Maintained by **Ravi Sharma** (ravisharmacs09@gmail.com)

## Suggested Learning Path

1. **Foundations** → Python (01), Databases (07)
2. **LLM Basics** → OpenAI/Claude API (03), AI Frameworks (02)
3. **Core AI Eng** → RAG (04), Vector DB (05), AI Agents (06)
4. **Production** → MLOps (08), Cloud (09), Security (11), Evaluation (12)
5. **Mastery** → AI System Design (10), GenAI Ecosystems (13)
6. **Prove it** → AI Real Projects (14)

## How to Use This Repo
- Work through folders in order, or jump to your weak areas.
- For each topic: read concepts → answer interview questions → complete hands-on tasks.
- Build and document projects in `14-AI-Real-Projects` as you learn.
- Revisit interview questions before interviews.
